# Cortado (Temporary Codename)

## SetUp
To get started you'll need [node](http://nodejs.org), and [npm](http://npmjs.org/)

### NPM

Install all packages
`npm install`

### Running
Run the server with node-dev
`node-dev runner.js`

## Client Side Stack
